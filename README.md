# 今日含金量 · Au

> 实时查看大盘金价与各大品牌足金零售价，附回收参考与换算器。

![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-4285F4?logo=googlechrome&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green)

---

## 功能

- **实时大盘金价** — AU9999 上海黄金交易所现货价，含涨跌幅
- **大盘回收基准价** — 门店回收参考（大盘价 − 10~30 元/克）
- **各大品牌零售价** — 周大福、老凤祥、菜百珠宝、中国黄金、周生生，标注最高/最低价
- **足金价格计算器** — 支持克/钱/两换算，选品牌即时计算总价
- **深色/浅色主题** — 跟随系统或手动切换
- **自动刷新** — 每 30 分钟后台更新，打开即显示最新数据

## 安装

### 从源码安装（开发者模式）

1. 下载或克隆本仓库

   ```bash
   git clone https://github.com/cw0121/today-gold-content.git
   ```

2. 打开 Chrome，地址栏输入 `chrome://extensions`

3. 右上角开启 **开发者模式**

4. 点击 **加载已解压的扩展程序**，选择本仓库文件夹

5. 扩展图标出现在工具栏，点击即用

---

## 使用

| 操作 | 说明 |
|------|------|
| 点击扩展图标 | 打开价格面板 |
| ↻ 刷新按钮 | 手动获取最新价格 |
| ☀️ / 🌙 | 切换浅色/深色主题 |
| 价格计算器 | 展开后输入重量，选择品牌和单位，自动计算总价 |
